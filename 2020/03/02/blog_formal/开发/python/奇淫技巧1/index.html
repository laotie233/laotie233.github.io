1、数字分割符

​ 为了提高数字的的可读性，Python 3.x 允许使用下划线_作为数字（包括整数和小数）的分隔符。通常每隔三个数字添加一个下划线，类似于英文数字中的逗号。下划线不会影响数字本身的值

4、长字符串 && 原始字符串

Python 长字符串由三个双引号"""或者三个单引号'''包围，也是多行注释的方法

python对长字符串的机制

  • 在长字符串中放置单引号或者双引号不会导致解析错误
  • 长字符串没有赋值给任何变量，那么这个长字符串就不会起到任何作用，和一段普通的文本无异，相当于被注释
  • Python 解释器并不会忽略长字符串，也会按照语法解析，只是长字符串起不到实际作用而已
  • 当程序中有大段文本内容需要定义成字符串时，优先推荐使用长字符串形式，非常强大，可以在字符串中放置任何内容，包括单引号和双引号
  • 长字符串中的换行、空格、缩进等空白符都会原样输出
